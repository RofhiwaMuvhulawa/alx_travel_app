Explanation of Dependencies
django: The core Django framework for building the web application.
djangorestframework: For developing RESTful APIs.
django-cors-headers: To handle Cross-Origin Resource Sharing (CORS) for API requests.
drf-yasg: For generating Swagger/OpenAPI documentation for the APIs.
celery: For handling asynchronous task queuing (to be configured in future milestones).
mysqlclient: The MySQL database adapter for Django to connect to the MySQL database.
django-environ: For managing environment variables securely using .env files.
